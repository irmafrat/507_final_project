Recent protest movement in Puerto Rico

This application allows you to search and retrieve embed tweets related to the recent Puerto Rico protests movements in 2017 and 2019.

The application is using the Documenting The Now Tweet Catalog to retrieve around 4,000 tweets from two different data sets.

The data sets used to create this project are produced by Joel Blanco-Rivera using the Documenting the Now Tools and TWARC command line program. The datasets are the following:

  • RickyRenuncia
  • #HuelgaUPR tweets, April 20 - May 18 2017
